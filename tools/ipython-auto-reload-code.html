ipython 自动加载代码 

In [1]: %load_ext auto-reload

In [2]: %autoreload 2

In [3]: from foo import some_function

In [4]: some_function()
Out[4]: 42

In [5]: # 编辑 foo.py 使其 return 43

In [6]: some_function()
Out[6]: 43

具体的看文档： http://ipython.org/ipython-doc/2/config/extensions/autoreload.html
